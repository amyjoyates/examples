This project holds a couple of examples.

1.  Generate all possible strings using ‘0’ and ‘1’ for a given width w
2.  Generate palindrome strings


In order to execute this:


1.  Clone repo
2.  You need Java 8 installed and set to JAVA_HOME
3.  Should be able to run: mvn test -DsuiteXmlFile=testng.xml (from the dir with the pom file)
